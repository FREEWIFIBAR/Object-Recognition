# Object-Recognition

Este proyecto implementa un proceso completo de reconocimiento de objetos en MATLAB, que incluye:
- Segmentación de imágenes
- Extracción de características
- Estandarización de características
- Selección de características
- Entrenamiento de múltiples clasificadores
- Evaluación cualitativa y cuantitativa

El sistema reconoce objetos industriales a partir de imágenes en escala de grises mediante técnicas clásicas de aprendizaje automático.
